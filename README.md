1. Introducción

Este documento detalla la estructura y contenido de un proyecto web diseñado para cumplir con los siguientes requisitos:

Ofrecer información general sobre un tema de interés.

Facilitar la navegación entre secciones temáticas.

Proporcionar una forma de contacto sencilla.

Presentar información académica y personal.

2. Descripción del Contenido del Sitio

El sitio web consta de las siguientes secciones principales:

2.1 index.html: Página Principal

La página principal incluye:

Una breve descripción del propósito del sitio.

Enlaces a las demás secciones.

Un diseño atractivo y responsivo para dar la bienvenida al usuario.

2.2 contact.html: Sección de Contacto

Contiene:

Un formulario sencillo con campos para:

Nombre

Correo electrónico

Mensaje

Nota: El formulario no incluye funcionalidad de envío o almacenamiento de datos.

2.3 net.html: Sección de Red

Proporciona:

Una lista de enlaces a las páginas de compañeros.

Información básica sobre cada página (nombre y temática).

2.4 topic.html: Sección Temática

Incluye:

Información detallada sobre un tema elegido, ya sea relacionado con el trabajo práctico o un interés personal.

Imágenes y enlaces relacionados con el tema.

2.5 degree.html: Sección de Grado

Describe:

El propósito y estructura del programa académico.

Una lista completa de las materias, cada una con una breve descripción.

Un enlace específico a fii.html, la página dedicada a "Fundamentos de la ingeniería informática".

2.6 fii.html: Fundamentos de la Ingeniería Informática

Explora:

Contenido detallado de la materia.

Objetivos y resultados de aprendizaje.

Recursos complementarios (como bibliografía o enlaces).

2.7 about.html: Acerca de Mí

Incluye:

Una breve descripción personal basada en las unidades 2 y 3 del Módulo 1.

Un currículum vitae con secciones estándar (educación, experiencia, habilidades, etc.).

Dos imágenes representativas.

Enlaces a redes sociales personales.

4. Detalles de Implementación

Lenguajes Utilizados

HTML5 para la estructura de las páginas.

CSS3 y Bootstrap para el diseño y la presentación.

JavaScript para animación del header

Consideraciones Técnicas

Uso de etiquetas semánticas para mejorar la accesibilidad.

Diseño responsivo utilizando media queries en CSS.

5. Estilo y Diseño

La paleta de colores es minimalista y profesional.

Uso de fuentes web modernas para mejorar la legibilidad.

Imágenes optimizadas para tiempos de carga rápidos.

6. Pruebas Realizadas

Navegación entre páginas para verificar enlaces.

Pruebas de diseño responsivo en dispositivos móviles y de escritorio.

Validación de código HTML y CSS con herramientas en línea.

7. Conclusión

Este proyecto demuestra la creación de un sitio web estructurado y funcional que integra múltiples secciones informativas y temáticas. Aunque el formulario de contacto es estático, cumple con los requisitos especificados. El diseño responsivo asegura una experiencia de usuario óptima en diferentes dispositivos. -->
